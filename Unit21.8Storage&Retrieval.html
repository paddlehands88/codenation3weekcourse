Unit 2 1.8 Storing and retrieving files effectively

From the perspective of the website, files such as images and icons, all need to be stored within an organised filing system. This is to make it easy when creating the website on VS Code for instance, for referencing images. At present all my images are located in the same file root as my homepage index.html. This has already become very cluttered and for future websites I will likely create separate image files possibly referenced by page to maintain organisation.

The css/styles.css page should have the css folder located at the same level as the index.html homepage.
